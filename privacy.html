Privacy Policy

This extension ("BOTC Event Recorder") does not collect, store, or transmit any personal data.

Data Usage

The extension accesses game event data from the Blood on the Clocktower web application currently open in the user's browser. This data is already available to the user as part of the game session.

Processing

All data processing occurs locally within the user's browser. The extension does not send any data to external servers.

User Control

The extension only records data when explicitly enabled by the user. Users can choose between manual mode, where recording is started and stopped directly, and auto mode, where recording begins automatically when a game session is detected and stops when the session ends. In both modes, the user retains full control and can disable recording at any time. Recorded data is exported only when the user chooses to download it, and it is saved locally to the user's device. No data is transmitted or stored externally.

Third Parties

No data is shared with third parties.
